# Patches

Diff and Merge

## Getting started

Run:

```console
$ cargo tauri dev
```

### Create project

Run:

```consoele
$ cargo create-tauri-app patches
```

and then run:

```console
$ bun install
$ cargo tauri dev
```

Optionally, modify `src-tauri/tauri.conf.json`.
